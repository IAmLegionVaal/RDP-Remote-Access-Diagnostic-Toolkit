# Remote Connectivity Diagnostic Toolkit

A read-only PowerShell toolkit for Windows connectivity context reporting.

## Features

- Service context
- Firewall profile context
- Network listener summary
- Recent event summary
- CSV, JSON, and HTML reports

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\RDP_Remote_Access_Diagnostic_Toolkit.ps1
```

## Safety

Diagnostic-only. It reports local system context for support review.
